GET STARTED

Three steps to smoother

1 · Install a loader

Grab Fabric (lightweight, great for performance mods) or Forge/NeoForge. Most mods here run on Fabric.

Pick one and stick with it
2 · Drop in mods

Download the .jar files and place them in your /mods folder. Match every mod to your game version.

Check version compatibility
3 · Add the library

Many Fabric mods need Fabric API. Forge mods may need a shared library — read each mod's page.

Launch and enjoy
